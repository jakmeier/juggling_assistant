## Git LFS (For handling large files in git)

### Install git lfs (Debian / Ubuntu)
```bash
curl -s https://packagecloud.io/install/repositories/github/git-lfs/script.deb.sh | sudo bash
sudo apt-get install git-lfs
git-lfs install
```
